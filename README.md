 # Решения домашних заданий по курсу "Многопоточное программирование на С++".

Директориям соответствуют следущие задания:
* ***task2*** - задание для лекции по теме "Процессы и потоки"

Каждая директория состоит из нескольких файлов:
* _**task.md**_ - описание задания
* _**solution.md**_ - описание решения, инструкции по запуску и демонстрация работы
* прочие _**.cpp**_ файлы - элементы решения

Решения получены на вирутальной системе с характеристиками:
* оперативная память 8 Гб
* процессоров 3
С этим связаны некоторые закономерности результатов.
